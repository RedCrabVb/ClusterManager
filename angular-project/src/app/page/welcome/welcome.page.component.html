CM

Cluster manager is a software for managing services in a distributed infrastructure

Abbreviations

Host is a representation of a physical or virtual machine. The host must be accessible via SSH, and it does not matter whether it is physical (hardware) or virtual (cloud).
Service is software running on hosts. A service can be either a single entity or a logical entity that combines several components (for example, in the case of HDFS).
Cluster is a set of services running on a set of hosts
InitFiles is a CM object that is needed to install and deploy services.
